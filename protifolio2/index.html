Conhecimentos

HTML

HTML (Linguagem de Marcação de HiperTexto), é utilizado para escrever o conteúdo dentro de um site utilizando marcaçoes, mais conhecidas como Tags.

CSS

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo, utilizada para aplicar cores, posicionamentos, fontes, entre outros estilos ao conteúdo do site, ou seja, o documento HTML

JavaScript

JavaScript, é a primeira linguagem de programação aprendida por qualquer desenvolvedor web, é usada para gerar interações do usuario com o site, junto com HTML e CSS é possível desenvolver qualquer Website. Com Javascript é também possível desenvolver sistemas back-end, e até jogos, sendo possivelmente a maior e mais utilizada linguagem de programação.

React.js

O React é uma biblioteca JavaScript criada pela META para o desenvolvimento de aplicações front-end tornando mais simples e rápida a criação de sites complexos. É usado para renderizar apenas os componentes necessários, aumentando a produtividade e diminuindo o tempo de carregamento. É provavelmente a biblioteca de JavaScript mais utilizada por desenvolvedores
